# ResumeCraft: Professional Resume Builder

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Visit-blue?style=for-the-badge&logo=vercel)](https://resume-craft-tawny.vercel.app/)

[![TypeScript](https://img.shields.io/badge/TypeScript-5.8.3-3178C6?style=flat&logo=typescript)](https://www.typescriptlang.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind%20CSS-3.4.17-38B2AC?style=flat&logo=tailwind-css)](https://tailwindcss.com/)

A modern, user-friendly web application for creating professional resumes with ease. Built with React, TypeScript, and Tailwind CSS, featuring multiple customizable templates and instant PDF export.

## ✨ Features

- **Multiple Templates**: Choose from 4 professionally designed resume templates (Classic Blue, Modern Maroon, Creative Purple, Minimal Classic)
- **Intuitive Form Interface**: Step-by-step forms for personal information, work experience, education, and skills
- **Live Preview**: Real-time preview of your resume as you fill in details
- **PDF Export**: Download your resume as a high-quality PDF with one click
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Type-Safe Development**: Built with TypeScript for reliability and maintainability
- **Accessible UI**: Uses shadcn/ui components for inclusive design

## 🛠️ Tech Stack

### Frontend Framework

- **React 18** - Component-based UI library
- **TypeScript** - Type-safe JavaScript
- **Vite** - Fast build tool and development server

### Styling & UI

- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Modern UI components built on Radix UI
- **Lucide React** - Beautiful icons

### State Management & Forms

- **Redux Toolkit** - Efficient state management
- **React Hook Form** - Performant forms with validation
- **Zod** - Schema validation

### Utilities

- **html2canvas** - HTML to canvas conversion for PDF generation
- **jsPDF** - PDF document generation
- **React Router** - Client-side routing
- **React Query** - Data fetching and caching

## 🚀 Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/resume-craft.git
   cd resume-craft
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   # or
   bun install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   bun dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to view the application.

## 📖 Usage

1. **Select a Template**: Choose from available resume templates on the home page
2. **Fill Details**: Complete the step-by-step forms:
   - Personal Information
   - Work Experience
   - Education
   - Key Skills
3. **Preview**: View your resume in real-time
4. **Download**: Export as PDF and save to your device
